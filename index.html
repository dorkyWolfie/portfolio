NATIONAL PARKS MACEDONIA | WEB DESIGN & DEVELOPMENT

Tools Used:

  1. Design: Adobe Photoshop, Figma
  2. Development: HTML5, CSS3, JavaScript

Project Name

National Parks of Macedonia

Project Description

National Parks Macedonia is a project created to promote Macedonian national parks. The idea was to create a website and rank it on Google.

Job Description

  • Create a fully functional website with optimized SEO in order to attract traffic from Google search.
  • Objectives
    1. Website Design
    2. Website Development
    3. On-page + Technical SEO
  • Process
    1. Research competitors, target group and inspiration
    2. Collect ideas, create sketches
    3. Filter everything (good and bad ideas) and create wireframes
    4. Create designs
    5. Create mockups
    6. Analyze and implement white-hat SEO tactics

Results

The website generated traffic in its first month being live. Increased engagement rate by 22.12%

EXTRA SHOP MK | ECOMMERCE & SOCIAL MEDIA

Tools Used:

  1. Design: Adobe Photoshop, Premiere Pro
  2. Development: WordPress, Yoast SEO

Brand Name

Extra Shop MK

Brand Description

Extra Shop MK is a new ecommerce brand that wants to be known for selling quality goods at affordable prices.

Job Description

  • Create an ecommerce website that is easy to use, and approchable to younger as well as to older customers.
  • I worked as a part of a 3 person team. My main objective was to provide a good copy for the products. My sidequests were to find or create and edit photos of the products, find and edit videos of people using the products, test and find any errors or mistakes in the build, create videos with voiceover for social media, manage social media accounts.
  • Objectives
    1. Generate Product Description and Details
    2. Find/Edit Product Photos
    3. Find/Edit Product Videos
    4. Test New Features
    5. Create Voiceover Videos

Results

We managed to go from 0 to more than 50 orders daily in 2-3 months.
